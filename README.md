<h1>
  Principal Objetivo
</h1>

<p>
  Alem de claro, praticar objetos, fiz esse login basico, para podermos ter uma noção minima de como funciona um sistema de Login, de guardar os dados do usuario, de proteger
  os dados, de jeito nenhum dados pessoais devem ter acesso publico, e nesse projeto eu mostro a pratica certa disso.
</p>

<p>
  Isso é um login basico, mas que de certa forma resume pra uma visão iniciante como funciona, temos registro, armazenamento de dados, privação de dados, validação de dados,
  basicante os conceitos fundamentais que giram em torno de um login.
</p>
